# Attack Simulation Lab

## Objective
This project focused on simulating real-world attack behavior using Kali Linux to test the security of an Ubuntu system, specifically targeting SSH authentication.

---

### Skills Learned
- Basic offensive security techniques
- SSH brute-force attack execution
- Understanding attacker methodologies
- Testing system vulnerabilities in a controlled environment

---

### Tools Used
- Kali Linux
- Hydra (or SSH brute-force tools)
- Ubuntu Linux
- SSH

---

## Steps

*Ref 1: Target Identification*  
Identified Ubuntu machine IP address.

*Ref 2: Attack Execution*  
Launched brute-force attack against SSH service.

*Ref 3: Attack Monitoring*  
Observed login attempts and responses.

*Ref 4: Result Analysis*  
Reviewed success/failure of attack attempts.
